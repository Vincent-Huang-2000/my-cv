MBTI Chat 是一个基于 16 种 MBTI 人格的全栈 AI 对话 Web App：用户可选择不同人格进行单人聊天，或最多 5 个角色参与群聊，并支持正反方“辩论模式”（自动轮换/手动点名发言、辩题分析与生成立场）。后端通过动态 Prompt Builder 将“人格设定 + 场景（学习/工作/情感等）+ 个性化配置（职业/情绪/星座/阶段）+ 对话上下文”组合成稳定的系统提示词，提升一致性与可控性；同时引入访问会话、心跳与事件埋点，沉淀访问量/消息量/人格偏好等运营数据。

- 核心功能：单人/群聊/辩论三种对话模式，角色选择与个性化设定，消息流式体验（自动滚动、加载态、移动端适配）
- Prompt 设计：人格锚定优先级、场景约束、群聊去复述与意图摘要，减少“同质化输出/互相引用”
- 数据分析：会话 Visit + Heartbeat 停留时长、选择/配置事件日志、全局与人格统计
- 技术栈：React + Vite + TailwindCSS；Node.js + Express；Prisma + SQLite（better-sqlite3）；OpenAI/DeepSeek API；Nginx/PM2 部署支持
